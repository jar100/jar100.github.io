java 공부

  • 레벨2 를 위해 자바 공부

패키지(Package)

  • 패키지(package)란 서로 관련있는 클래스 또는 인터페이스 들을 묶어 놓은 묶음이다.
  • 패키지 이름은 폴더.폴더.폳러
  • 도메인 이름이 abc.com 프로젝트 이름이 javastudy 이라면 com.abc.javastudy.Hello 로 패키지를 지정 할 수 있다.

함수(Method)

  • 기본 형태

  • static 정적변수는 뭐지???

    • 전역변수에 붙으면 지역변수로 바꾼다.
    • 지역변수에 붙으면 소멸시기가 프로그램 종료시로 바뀐다.
  • String[] arg 이건 뭘까??
    • main 메소드의 매개 변수
  • void 메소드가 값을 반환하지 않음을 의미한다.

Integer 와 int 의 차이

  • integer Wrapper 클래스 (객체)
    • Unboxing 을 하지 않으면 산술 연산이 불가능 하지만, null 값을 처리할 수 있다.
    • null 값 처리가 용이하기 때문에 SQL과 연동할 경우 처리가 용이하다.
  • int
    • int 는 null 로 초기화 할수없다.
