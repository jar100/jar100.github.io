변수

데이터(자료)를 임시로 담을 수 있는 상자(메모리 공간에 저장)
변수 값은 2진수로 변환되 저장된다.

자료형

기본 자료형 , 객체자료형

  • 기본자료형 데이터가 변수에 직접 저장된다.

    • 정수형: char, int, long
    • 실수형: float, double
    • 논리형: boolean
    • 변수의 크기가 다 다르다.

  • 객체자료형 객체 메모리 주소가 변수에 저장된다.

    • String
