  4. 스텍 메모리 힙메모리

  5. JUnit
    대표적인 단정문

  • assertArrayEquals(a,b) : 배열 a와b가 일치함을 확인
  • assertEquals(a,b) : 객체 a와b의 값이 같은지 확인
  • assertSame(a,b) : 객체 a와b가 같은 객체임을 확인
  • assertTrue(a) : a가 참인지 확인
  • assertNotNull(a) : a객체가 null이 아님을 확인
  • @Test(timeout=5000) : 실행시간을 제한할 수 있다.

피드백
