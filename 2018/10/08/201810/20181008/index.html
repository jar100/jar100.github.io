사용이유 모듈의 재 사용성이 증가한다.
유지보수가 쉽고 비용이 절감된다.

  1. 모듈의 단위태스트를 지향하자.

DTO 와 VO는 다르다.

DTO 는 setter, getter 허용

VO (value object) 변경 불가능
value = 변경 불가 (mutable object)
variable = 변경 가능 (immutable object)

뮤터블 set 들어가면됨
임뮤터블 string 예로 들자.

자료구조를 많이 쓴다.
데이터를 어떤형태로 저장하고 사용할것이냐
자바는 자료구조는 콜렉션 프레임 워크로 사용된다.
list set map
ArrayList > HashMap > HashSet
3가지만 잘 사용하자.

  1. ArrayList는 배열에 쓰면됨
  2. Map 은 key value
